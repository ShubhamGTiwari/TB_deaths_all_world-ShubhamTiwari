# TB_deaths_all_world-ShubhamTiwari
This repository contains the files for my Personal Project on deaths caused by Tuberculosis (TB).

## Project Description
The primary goal of this project is to work with real data to analyze a real-world data. This programme covers all of the essential skills of a data analytics, including: -Assessment of Data Quality -Insights from Data -Insights and Presentation of Data.
### Methods used
- Data Cleaning
- Data Exploration
- Data Modeling
- Data Visualisation

### Technologies used
- Python
- MS Excel
- Jupyter Notebook
- Pandas
